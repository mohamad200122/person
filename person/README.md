# Flutter Personal Blog (Demo)

Simple one-page personal blog website for a Flutter mobile developer.
Built with plain HTML, CSS, and JavaScript to test free hosting deployment.

## Features

- Single-page layout (Hero, About, Skills, Projects, Blog, Contact)
- Arabic and English language switch
- RTL/LTR direction switch based on language
- Responsive design for mobile and desktop
- Lightweight and easy to deploy

## Run Locally

1. Open `index.html` directly in your browser.
2. Or use VS Code Live Server extension for local preview.

## Deploy to GitHub Pages

1. Create a new GitHub repository.
2. Upload project files (`index.html`, `README.md`, and any assets you add later).
3. Go to repository `Settings` -> `Pages`.
4. In `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main` (root)
5. Save, then wait 1-2 minutes.
6. Open the generated GitHub Pages URL.

## Customize Before Publishing

- Replace placeholder social links in the Contact section.
- Update email address.
- Edit demo post/project texts with your real content.
- Optionally add a custom domain in GitHub Pages settings.
